## Usage

1. Clone the repo
    ```
    git clone https://github.com/YourInternetPal/sample-nft-listing-with-moralis.git
    ```
2. Install Dependencies
    ```
        yarn install
    ```
3. Get a copy of the .env.example file and rename it as .env file. Then update the variables
4. Run the project
    ```
        yarn dev
    ```
